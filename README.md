## To deploy this app

1. Clone the repo & install the dependencies

```sh
git clone git@github.com:dabit3/conference-app-in-a-box.git

cd conference-app-in-a-box

npm install
```

2. Initialize and deploy the Amplify project

```sh
amplify init

amplify push
```
